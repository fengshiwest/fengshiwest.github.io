### Ubuntu 调节亮度

、、、
sudo add-apt-repository ppa:apandada1/brightness-controller
sudo apt-get update
sudo apt-get install brightness-controller-simple
、、、

安装完成后，输入`brightness-controller-simple`即可调节，每次开机需要重新调节。
